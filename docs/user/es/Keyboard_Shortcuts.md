# Atajos de teclado

Stand: 2026-07-15

Pulsa `F1` (o **Paleta de comandos → Mostrar atajos de teclado**) para abrir en cualquier momento la ventana de atajos. Agrupa todos los atajos y gestos del ratón en zonas entre las que cambias con los chips de arriba, tiene un campo de búsqueda que abarca todas las zonas, y detecta tu plataforma automáticamente: muestra `Ctrl`/`Alt` en Windows y Linux, y `⌘`/`⌥` en macOS. Por eso, en las tablas de abajo `Ctrl` significa `⌘` en macOS y `Alt` significa `⌥`.

## General

| Atajo | Acción |
|---|---|
| `Ctrl+P` (o `Ctrl+Shift+P`) | Abrir la paleta de comandos |
| `Ctrl+O` | Selector rápido – buscar y abrir archivos |
| `Ctrl+,` | Abrir configuración |
| `Ctrl+Plus` / `Ctrl+Minus` | Ampliar / reducir el zoom de la interfaz (80–150 %) |
| `Ctrl+0` | Restablecer el zoom al 100 % |
| `Ctrl+S` | Confirmar guardado (Plainva guarda automáticamente) |
| `F1` | Mostrar esta ventana |
| `Esc` | Cancelar acción (cerrar diálogo, cancelar renombrado) |

## Notas y archivos

| Atajo | Acción |
|---|---|
| `Ctrl+N` | Nueva nota en la carpeta seleccionada |
| `Ctrl+Shift+D` | Abrir la nota diaria de hoy |
| `F2` | Renombrar la nota activa |
| `Ctrl+Alt+Izquierda` / `Ctrl+Alt+Derecha` | Atrás / Adelante (historial de navegación) |

En el árbol de archivos: **clic** selecciona y abre, `Ctrl`+clic alterna una selección múltiple, `Shift`+clic selecciona un rango, **clic central** abre un archivo en una pestaña nueva, **arrastrar** mueve archivos, **clic derecho** abre el menú contextual y `Supr` mueve la selección a la papelera.

## Vista, ventanas y pestañas

| Atajo | Acción |
|---|---|
| `Ctrl+E` | Alternar lectura ↔ edición |
| `Ctrl+Shift+E` | Alternar vista previa en vivo ↔ código fuente Markdown |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | Alternar la barra lateral izquierda / derecha |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | Dividir el editor lado a lado / apilado |
| `Ctrl+Shift+G` | Abrir el grafo / mapa del vault |
| `Ctrl+Shift+F` | Buscar y reemplazar en todo el vault |
| `Ctrl+T` | Nueva pestaña (elige una nota para abrir) |
| `Ctrl+W` | Cerrar pestaña |
| `Ctrl+Shift+T` | Reabrir la última pestaña cerrada |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Pestaña siguiente / anterior (`Ctrl` en todos los sistemas operativos) |
| `Ctrl+1` … `Ctrl+8` | Saltar a la pestaña 1–8 |
| `Ctrl+9` | Saltar a la última pestaña |

El modo de enfoque (ambas barras laterales ocultas) está en la paleta de comandos.

## Formatear texto

Funciona en el editor (vista previa en vivo o código fuente):

| Atajo | Acción |
|---|---|
| `Ctrl+B` / `Ctrl+I` | Negrita / cursiva |
| `Ctrl+Shift+S` | Tachado |
| `Ctrl+Shift+H` | Resaltado (`==marca==`) |
| `Ctrl+K` | Insertar un enlace / convertir la selección en un enlace |
| `Ctrl+Shift+1` / `2` / `3` | Encabezado 1 / 2 / 3 |
| `Ctrl+Shift+0` | Texto normal (quitar el encabezado) |
| `Ctrl+Enter` | Alternar la casilla de la tarea |

También puedes seleccionar texto para usar la barra flotante (negrita, cursiva, tachado, código, resaltado, enlace), y hacer clic en una casilla `[ ]` renderizada para alternar una tarea en cualquier modo.

## Insertar

Escribe esto directamente en el editor:

| Entrada | Acción |
|---|---|
| `/` | Menú de comandos (bloques, formato, plantillas, emoji) |
| `@` | Mención: fecha, enlace a una nota o base de datos |
| `[[` | Enlace interno – elige una nota |
| `![[` | Incrustar una nota o una imagen |
| `#` | Completar una etiqueta |
| `:name` | Emoji por nombre (inserta el carácter) |
| `Ctrl+Alt+T` | Insertar plantilla |

Soltar un archivo en el editor incrusta una imagen (o copia el archivo y lo enlaza si es de otro tipo).

## Grafo

| Atajo | Acción |
|---|---|
| `Ctrl+Shift+G` | Abrir el grafo / mapa del vault |
| Teclas de flecha | Mover el foco entre nodos |
| `Enter` | Abrir el nodo enfocado |

Ratón: **la rueda** hace zoom; **arrastre con el botón central** o `Ctrl`+arrastre desplaza el mapa (incluso sobre un nodo); **arrastrar en un espacio vacío** dibuja un lazo de selección (mapa del vault); `Alt`+arrastre mueve un nodo junto con sus vecinos enlazados (mapa del vault); **arrastrar un nodo sobre otro** crea un enlace o una relación de base de datos (mapa del vault); un solo clic abre (grafo de contexto/de base de datos) o selecciona (mapa del vault); **doble clic en una burbuja de carpeta** la despliega (doble clic en el anillo la contrae); `Ctrl`+clic abre en la división, **clic central** abre en una pestaña nueva; **clic derecho** abre el menú contextual; la **aguja de anclaje** (arriba a la derecha) alterna si las posiciones se recuerdan.

## Bases de datos (.base)

| Entrada | Acción |
|---|---|
| Clic en una celda | Editar in situ; una casilla se alterna directamente |
| Clic en una tarjeta / fila | Abrir la nota en una ventana de vista rápida |
| `Ctrl`+clic en una tarjeta | Abrir la nota en el panel dividido |
| Arrastrar una tarjeta | Tablero: define el valor de la columna; calendario/cronología: define la fecha |
| Arrastrar el encabezado de una columna | Reordenar columnas (y las opciones de la propiedad) |
| Clic en el encabezado de una tabla | Cambiar el orden |
| Clic / arrastre en una pestaña de vista | Cambiar de vista; arrastrar para reordenar; `▾` renombra/duplica/elimina |
| **+ Entrada** | Nueva entrada con la plantilla predeterminada (`▾` = plantilla/carpeta) |

## Ratón y gestos (en todas partes)

- **Control de bloque (`⠿`):** toca para abrir el menú del bloque (convertir en, duplicar, mover, eliminar); arrastra para reordenar.
- **Pestañas:** arrastra para reordenar, entre paneles, o hasta un borde para dividir.
- **Enlaces:** haz clic en un enlace wiki o Markdown para abrirlo (`Ctrl`+clic para una pestaña nueva).
- **Clic derecho** en una selección o un campo abre el menú propio de Plainva para copiar / cortar / pegar. `F5` y `Ctrl+R` **no** recargan (perderían las pestañas y los búferes sin guardar).
- **Ventana de vista rápida:** arrastra el encabezado para moverla, el asa de la esquina para cambiar el tamaño; `◀ ▶` tiene su propio historial.
- **Calendario:** haz clic en el nombre del mes para elegir mes/año; haz clic en un día para su nota diaria.
- **Editor de imágenes:** dibuja con el ratón (lápiz/flecha/rectángulo/texto/recorte); haz zoom con los botones.
