# Atajos de teclado

Stand: 2026-07-05

Las combinaciones de teclas más importantes de un vistazo. Dentro de la app, `F1` (o **Configuración → Mostrar atajos de teclado**) muestra la misma lista en cualquier momento. En macOS, `⌘` sustituye a `Ctrl`.

## Globales

| Atajo | Acción |
|---|---|
| `Ctrl+O` o `Ctrl+K` | Selector rápido: buscar y abrir archivos |
| `Ctrl+P` | **Comandos** – abrir la paleta de comandos (escribe, Intro ejecuta) |
| `Ctrl+Alt+B` | Alternar la barra lateral izquierda |
| `Ctrl+Alt+R` | Alternar la barra lateral derecha |
| `Ctrl+,` | Abrir configuración |
| `F1` | Mostrar atajos de teclado |
| `Esc` | Cancelar acción (cerrar diálogo, cancelar renombrado) |

## Editor

| Atajo | Acción |
|---|---|
| `/` (al principio de una línea) | Abrir el menú de formato (menú de barra oblicua) |
| `@` (dentro del texto) | Insertar una fecha, un enlace a nota o una base de datos |
| `Ctrl+F` | Buscar y reemplazar en la nota actual |
| `Ctrl+Alt+T` | Insertar plantilla |
| `Ctrl+Alt+V` | Dividir editor: lado a lado |
| `Ctrl+Alt+S` | Dividir editor: apilado |

## Árbol de archivos

| Entrada | Acción |
|---|---|
| Clic | Seleccionar |
| `Ctrl`+clic | Añadir a la selección / quitar de la selección |
| `Shift`+clic | Seleccionar un rango |
| Clic central | Abrir en una nueva pestaña |

## Bases de datos (.base)

| Entrada | Acción |
|---|---|
| Clic en una celda/valor | Editar directamente (edición en línea) |
| Clic en el título de una entrada | Abrir en la ventana de vista previa |
| `Ctrl`+clic en una entrada | Abrir en panel dividido |
