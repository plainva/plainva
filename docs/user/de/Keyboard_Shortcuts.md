# Tastenkürzel

Stand: 2026-07-10

Die wichtigsten Tastenkombinationen im Überblick. In der App zeigt `F1` (oder **Einstellungen → Tastenkombinationen anzeigen**) jederzeit dieselbe Liste. Auf macOS gilt `⌘` statt `Strg`.

## Global

| Kürzel | Aktion |
|---|---|
| `Strg+O` oder `Strg+K` | Schnellwechsel – Dateien suchen und öffnen |
| `Strg+P` | **Befehle** – die Befehls-Palette öffnen (tippen, Enter führt aus) |
| `Strg+Alt+B` | Linke Seitenleiste ein-/ausblenden |
| `Strg+Alt+R` | Rechte Seitenleiste ein-/ausblenden |
| `Strg+,` | Einstellungen öffnen |
| `Strg+Plus` / `Strg+Minus` | Oberflächen-Zoom vergrößern/verkleinern (80–150 %) |
| `Strg+0` | Oberflächen-Zoom zurücksetzen |
| `F1` | Tastenkombinationen anzeigen |
| `Esc` | Aktion abbrechen (Dialog schließen, Umbenennen abbrechen) |

## Editor

| Kürzel | Aktion |
|---|---|
| `/` (am Zeilenanfang) | Formatierungs-Menü (Slash-Menü) öffnen |
| `@` (im Text) | Datum, Notiz-Link oder Datenbank einfügen |
| `Strg+F` | Suchen & Ersetzen in der aktuellen Notiz |
| `Strg+Alt+T` | Vorlage einfügen |
| `Strg+Alt+V` | Editor teilen – nebeneinander |
| `Strg+Alt+S` | Editor teilen – übereinander |

## Dateibaum

| Eingabe | Aktion |
|---|---|
| Klick | Auswählen |
| `Strg`+Klick | Auswahl einzeln erweitern/verkleinern |
| `Umschalt`+Klick | Bereich auswählen |
| Mittelklick | In neuem Tab öffnen |

## Datenbanken (.base)

| Eingabe | Aktion |
|---|---|
| Klick auf Zelle/Wert | Direkt bearbeiten (Inline-Editing) |
| Klick auf Eintragstitel | Im Peek-Fenster öffnen |
| `Strg`+Klick auf Eintrag | Im Split öffnen |
