# Tastenkürzel

Stand: 2026-07-15

Mit `F1` (oder **Befehlspalette → Tastenkombinationen anzeigen**) öffnest Du jederzeit das Tastenkürzel-Fenster. Es gruppiert alle Kürzel und Maus-Gesten in Bereiche, zwischen denen Du oben mit den Chips wechselst, hat ein Suchfeld über alle Bereiche und erkennt Deine Plattform automatisch: `Strg`/`Alt` unter Windows und Linux, `⌘`/`⌥` unter macOS. In den Tabellen unten steht `Strg` daher für `⌘` unter macOS und `Alt` für `⌥`.

## Allgemein

| Kürzel | Aktion |
|---|---|
| `Strg+P` (oder `Strg+Umschalt+P`) | Befehlspalette öffnen |
| `Strg+O` | Schnellwechsel – Dateien suchen und öffnen |
| `Strg+,` | Einstellungen öffnen |
| `Strg+Plus` / `Strg+Minus` | Oberfläche vergrößern / verkleinern (80–150 %) |
| `Strg+0` | Zoom auf 100 % zurücksetzen |
| `Strg+S` | Speichern bestätigen (Plainva speichert automatisch) |
| `F1` | Dieses Fenster anzeigen |
| `Esc` | Aktion abbrechen (Dialog schließen, Umbenennen abbrechen) |

## Notizen & Dateien

| Kürzel | Aktion |
|---|---|
| `Strg+N` | Neue Notiz im gewählten Ordner |
| `Strg+Umschalt+D` | Heutige Tagesnotiz öffnen |
| `F2` | Aktive Notiz umbenennen |
| `Strg+Alt+Links` / `Strg+Alt+Rechts` | Zurück / Vor (Navigationsverlauf) |

Im Dateibaum: **Klick** wählt aus und öffnet, `Strg`+Klick schaltet eine Mehrfachauswahl um, `Umschalt`+Klick wählt einen Bereich, **mittlere Maustaste** öffnet eine Datei in neuem Tab, **Ziehen** verschiebt Dateien, **Rechtsklick** öffnet das Kontextmenü.

## Ansicht, Fenster & Tabs

| Kürzel | Aktion |
|---|---|
| `Strg+E` | Lesen ↔ Bearbeiten umschalten |
| `Strg+Umschalt+E` | Live-Vorschau ↔ Markdown-Quelltext umschalten |
| `Strg+Alt+B` / `Strg+Alt+R` | Linke / rechte Seitenleiste umschalten |
| `Strg+Alt+V` / `Strg+Alt+S` | Editor nebeneinander / übereinander teilen |
| `Strg+Umschalt+G` | Graph / Vault-Karte öffnen |
| `Strg+Umschalt+F` | Im ganzen Vault suchen & ersetzen |
| `Strg+T` | Neuer Tab (Notiz zum Öffnen wählen) |
| `Strg+W` | Tab schließen |
| `Strg+Umschalt+T` | Zuletzt geschlossenen Tab wieder öffnen |
| `Strg+Tab` / `Strg+Umschalt+Tab` | Nächster / vorheriger Tab (`Strg` auf jedem System) |
| `Strg+1` … `Strg+8` | Zu Tab 1–8 springen |
| `Strg+9` | Zum letzten Tab springen |

Der Fokus-Modus (beide Seitenleisten aus) liegt in der Befehlspalette.

## Text formatieren

Gilt im Editor (Live-Vorschau oder Quelltext):

| Kürzel | Aktion |
|---|---|
| `Strg+B` / `Strg+I` | Fett / kursiv |
| `Strg+Umschalt+S` | Durchgestrichen |
| `Strg+Umschalt+H` | Hervorheben (`==Markierung==`) |
| `Strg+K` | Link einfügen / Auswahl in Link umwandeln |
| `Strg+Umschalt+1` / `2` / `3` | Überschrift 1 / 2 / 3 |
| `Strg+Umschalt+0` | Normaler Text (Überschrift entfernen) |
| `Strg+Eingabe` | Aufgaben-Häkchen umschalten |

Du kannst außerdem Text markieren, um die schwebende Werkzeugleiste zu nutzen (Fett, Kursiv, Durchgestrichen, Code, Hervorheben, Link), und ein gerendertes `[ ]`-Häkchen anklicken, um eine Aufgabe in jedem Modus umzuschalten.

## Text bearbeiten

| Kürzel | Aktion |
|---|---|
| `Strg+Z` / `Strg+Y` (oder `Strg+Umschalt+Z`) | Rückgängig / Wiederherstellen |
| `Strg+A` | Alles auswählen |
| `Tab` / `Umschalt+Tab` | Einrücken / ausrücken |
| `Alt+Auf` / `Alt+Ab` | Zeile nach oben / unten verschieben |
| `Umschalt+Alt+Ab` | Zeile duplizieren |
| `Strg+Umschalt+K` | Zeile löschen |
| `Strg+D` | Nächstes gleiches Vorkommen zur Auswahl hinzufügen |
| `Strg+F` | In der Notiz suchen & ersetzen |
| `Strg+G` / `F3` | Weitersuchen (Umschalt = rückwärts) |
| `Strg+Alt+G` | Zu Zeile springen |
| `Strg+Umschalt+[` / `Strg+Umschalt+]` | Falten / entfalten (macOS: `⌘⌥[` / `⌘⌥]`) |
| `Strg+Leertaste` | Autovervollständigung auslösen |

## Einfügen

Direkt im Editor tippen:

| Eingabe | Aktion |
|---|---|
| `/` | Befehlsmenü (Blöcke, Formatierung, Vorlagen, Emoji) |
| `@` | Erwähnung: Datum, Notiz-Link oder Datenbank |
| `[[` | Wiki-Link – Notiz wählen |
| `![[` | Notiz oder Bild einbetten |
| `#` | Tag vervollständigen |
| `:name` | Emoji per Namen (fügt das Zeichen ein) |
| `Strg+Alt+T` | Vorlage einfügen |

Eine Datei ins Editorfeld zu ziehen bettet ein Bild ein (oder kopiert eine andere Datei herein und verlinkt sie).

## Graph

| Kürzel | Aktion |
|---|---|
| `Strg+Umschalt+G` | Graph / Vault-Karte öffnen |
| Pfeiltasten | Fokus zwischen Knoten bewegen |
| `Eingabe` | Fokussierten Knoten öffnen |

Maus: **Mausrad** zoomt; **mittlere Maustaste + ziehen** oder `Strg`+ziehen verschiebt die Karte (auch über einem Knoten); **Ziehen auf leerer Fläche** zieht ein Auswahlrechteck (Vault-Karte); `Alt`+ziehen verschiebt einen Knoten samt verbundener Nachbarn (Vault-Karte); **Knoten auf Knoten ziehen** erstellt eine Verbindung oder Datenbank-Relation (Vault-Karte); ein Einfachklick öffnet (Kontext-/Base-Graph) bzw. wählt aus (Vault-Karte); **Doppelklick auf eine Ordner-Blase** entfaltet sie (Doppelklick auf den Ring klappt sie ein); `Strg`+Klick öffnet im geteilten Bereich, **mittlere Maustaste** in neuem Tab; **Rechtsklick** öffnet das Kontextmenü; die **Pin-Nadel** (oben rechts) schaltet um, ob Positionen gemerkt werden.

## Datenbanken (.base)

| Eingabe | Aktion |
|---|---|
| Klick auf Zelle | Direkt bearbeiten; ein Häkchen schaltet direkt |
| Klick auf Karte / Zeile | Notiz im Peek-Fenster öffnen |
| `Strg`+Klick auf Karte | Notiz im geteilten Bereich öffnen |
| Karte ziehen | Board: Spaltenwert setzen; Kalender/Zeitachse: Datum setzen |
| Spaltenkopf ziehen | Spalten umsortieren (und die Optionen der Eigenschaft) |
| Klick auf Tabellenkopf | Sortierung wechseln |
| Klick / Ziehen an der Ansichts-Pille | Ansicht wechseln; ziehen zum Umsortieren; `▾` = umbenennen/duplizieren/löschen |
| **+ Eintrag** | Neuer Eintrag mit Standardvorlage (`▾` = Vorlage/Ordner) |

## Maus & Gesten (überall)

- **Blockgriff (`⠿`):** antippen für das Block-Menü (Umwandeln, Duplizieren, Verschieben, Löschen); ziehen zum Umsortieren.
- **Tabs:** ziehen zum Umsortieren, zwischen Bereichen oder an den Rand zum Teilen.
- **Links:** einen Wiki-/Markdown-Link anklicken zum Öffnen (`Strg`+Klick für neuen Tab).
- **Rechtsklick** auf eine Auswahl oder ein Feld öffnet Plainvas eigenes Kopieren-/Ausschneiden-/Einfügen-Menü. `F5` und `Strg+R` laden **nicht** neu (das würde Tabs und ungespeicherte Puffer verwerfen).
- **Peek-Fenster:** Kopf ziehen zum Verschieben, Eck-Griff zum Anpassen der Größe; `◀ ▶` ist ein eigener Verlauf.
- **Kalender:** Monatsnamen anklicken für Monat/Jahr; einen Tag anklicken für seine Tagesnotiz.
- **Bild-Editor:** mit der Maus zeichnen (Stift/Pfeil/Rechteck/Text/Zuschneiden); Zoom über die Knöpfe.
