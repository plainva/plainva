# Google Drive同期の設定（Bring Your Own Credentials）

Plainvaでローカル保管庫をGoogle Driveと同期するには、自前のGoogle API認証情報（Credentials）を使用できます。Plainvaはまだ（今のところ）Googleの中央CASA検証を通過していないため、この**Bring Your Own Credentials（BYO）**方式は、あなたの個人ファイルを安全に同期する方法を提供します。

これは実質的に、あなただけが所有し、あなただけがアクセスできる自分専用の小さな「開発者プロジェクト」をGoogle上に構築する作業です。

## 手順ガイド

### 1. Google Cloud Consoleでプロジェクトを作成する
1. [Google Cloud Console](https://console.cloud.google.com/)にアクセスします。
2. Googleアカウントでログインします。
3. 左上（Google Cloudのロゴの隣）にあるプロジェクトのドロップダウンメニューを開き、**新しいプロジェクト**を選択します。
4. 名前を入力し（例:「Plainva Sync」）、**作成**をクリックします。

### 2. Google Drive APIを有効化する
1. 上部のドロップダウンで、新しく作成したプロジェクトを選択します。
2. 上部の検索バーで**Google Drive API**を検索し、「Marketplace」の下にある項目を選択します。
3. **有効にする**をクリックします。

### 3. OAuth同意画面を設定する
Plainvaがあなたの認証情報を使用するには、同意画面（「OAuth Consent Screen」）を設定する必要があります。使用するのはあなただけなので、これは「テスト」モードのままにします。

1. 左側のメニューの**APIとサービス**の下にある**OAuth同意画面**を開きます。
2. 「User Type」で**外部**を選択し（Google Workspaceを使用している場合を除く）、**作成**をクリックします。
3. **アプリ情報:**
   - アプリ名: 例:「Plainva」
   - ユーザーサポートメール: あなた自身のメールアドレス
   - デベロッパーの連絡先情報: あなた自身のメールアドレス
   - **保存して次へ**をクリックします。
4. **スコープ:**
   - **スコープを追加または削除**をクリックします。
   - `.../auth/drive`（Google Drive API、フルアクセス）を検索し、チェックボックスにチェックを入れます。
   - *背景: フルアクセスが必要なのは、あなたが同期フォルダーにGoogle Driveのウェブインターフェース経由で追加したファイルも、Plainvaが同期できるようにするためです。*
   - 更新をクリックし、次に**保存して次へ**をクリックします。
5. **テストユーザー:**
   - **ユーザーを追加**をクリックします。
   - 後でPlainvaの同期に使用するGoogleメールアドレスを正確に入力します。
   - **保存して次へ**をクリックし、ダッシュボードに戻ります。

*重要: ステータスは「テスト」のままにしてください。アプリを公開する必要はありません。テストモードでは、トークンは7日後に期限切れになります——Plainvaはバックグラウンドで自動的に更新しますが、大きな変更やスコープの変更後は、再ログインが必要になる場合があります。*

### 4. 認証情報（クライアントIDとシークレット）を作成する
1. 左側のメニューで**認証情報**を開きます。
2. 上部の**認証情報を作成**をクリックし、**OAuthクライアントID**を選択します。
3. 「アプリケーションの種類」として**デスクトップアプリ**（または「その他のUI」）を選択します。
4. 名前: 例:「Plainva Desktop Client」。
5. **作成**をクリックします。
6. ポップアップに**クライアントID**と**クライアントシークレット**が表示されます。

### 5. Plainvaに入力する
1. Plainvaを開き、保管庫の設定（該当する保管庫の歯車アイコン）に移動します。
2. **クラウド同期**セクションを開きます。
3. プロバイダーとして**Google Drive**を選択します。
4. コピーした**クライアントID**と**クライアントシークレット**を対応するフィールドに貼り付けます。
5. **Googleに接続**をクリックします。
6. Googleのブラウザーウィンドウが開きます。「テストユーザー」に追加したアカウントでログインします。
7. Googleが、アプリが未検証であるという警告を表示する場合があります。**詳細**をクリックし、次に**Plainvaに移動（安全ではないページ）**をクリックします。
8. 要求された権限を確認します。

これで、あなたの保管庫は自前の認証情報を通じてGoogle Driveと安全に同期されるようになりました。
