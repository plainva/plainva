## 基本原則

- **あなたのファイルはあなたのものです。** 保管庫は、Markdownファイルが入った普通のフォルダーです。いつでも他のプログラムで開いたり、コピーしたり、バックアップしたりできます。
- **純粋なMarkdownが正規の形式です。** プロパティ、アイコン、データベースといった追加機能も、オープンで読みやすいテキスト形式で保存されます。
- **Obsidianとの互換性があります。** 既存のObsidian保管庫が損なわれたり、再フォーマットされたりすることはありません。Obsidianは、Plainvaが作成するすべてのファイルを開くことができます。
