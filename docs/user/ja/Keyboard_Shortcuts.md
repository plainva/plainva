# キーボードショートカット

最終更新: 2026-07-15

`F1`（または**コマンドパレット → キーボードショートカットを表示**）を押すと、いつでもショートカットのウィンドウが開きます。ここでは、すべてのショートカットとマウスジェスチャーが上部のチップで切り替えるカテゴリにグループ分けされており、検索欄は全カテゴリを横断して機能し、プラットフォームも自動的に判定されます: WindowsとLinuxでは`Ctrl`/`Alt`、macOSでは`⌘`/`⌥`と表示されます。そのため以下の表の`Ctrl`はmacOSでは`⌘`を、`Alt`は`⌥`を意味します。

## 一般

| ショートカット | 動作 |
|---|---|
| `Ctrl+P`（または`Ctrl+Shift+P`） | コマンドパレットを開く |
| `Ctrl+O` | クイックスイッチャー——ファイルを検索して開く |
| `Ctrl+,` | 設定を開く |
| `Ctrl+Plus` / `Ctrl+Minus` | UI を拡大 / 縮小（80–150 %） |
| `Ctrl+0` | ズームを 100 % にリセット |
| `Ctrl+S` | 保存を確定（Plainvaは自動的に保存します） |
| `F1` | このウィンドウを表示 |
| `Esc` | 操作をキャンセル（ダイアログを閉じる、名前変更をキャンセル） |

## ノートとファイル

| ショートカット | 動作 |
|---|---|
| `Ctrl+N` | 選択したフォルダーに新規ノートを作成 |
| `Ctrl+Shift+D` | 今日のデイリーノートを開く |
| `F2` | アクティブなノートの名前を変更 |
| `Ctrl+Alt+←` / `Ctrl+Alt+→` | 戻る / 進む（ナビゲーション履歴） |

ファイルツリーでは、**クリック**で選択して開き、`Ctrl`+クリックで複数選択を切り替え、`Shift`+クリックで範囲を選択し、**中クリック**でファイルを新しいタブで開き、**ドラッグ**でファイルを移動し、**右クリック**でコンテキストメニューを開き、`Delete` で選択項目をゴミ箱に移動します。

## 表示・ウィンドウ・タブ

| ショートカット | 動作 |
|---|---|
| `Ctrl+E` | 閲覧 ↔ 編集を切り替え |
| `Ctrl+Shift+E` | ライブプレビュー ↔ Markdownソースを切り替え |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | 左 / 右サイドバーを切り替え |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | エディターを分割——左右 / 上下に並べる |
| `Ctrl+Shift+G` | グラフ / 保管庫マップを開く |
| `Ctrl+Shift+F` | 保管庫全体で検索と置換 |
| `Ctrl+T` | 新しいタブ（開くノートを選択） |
| `Ctrl+W` | タブを閉じる |
| `Ctrl+Shift+T` | 最後に閉じたタブを再度開く |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 次 / 前のタブ（`Ctrl`はすべてのOSで共通） |
| `Ctrl+1` … `Ctrl+8` | タブ 1〜8 に移動 |
| `Ctrl+9` | 最後のタブに移動 |

フォーカスモード（両方のサイドバーを非表示）はコマンドパレットにあります。

## テキストの書式設定

エディターで使えます（ライブプレビューまたはソース）:

| ショートカット | 動作 |
|---|---|
| `Ctrl+B` / `Ctrl+I` | 太字 / 斜体 |
| `Ctrl+Shift+S` | 取り消し線 |
| `Ctrl+Shift+H` | ハイライト（`==マーク==`） |
| `Ctrl+K` | リンクを挿入 / 選択範囲をリンクに変換 |
| `Ctrl+Shift+1` / `2` / `3` | 見出し1 / 2 / 3 |
| `Ctrl+Shift+0` | 通常のテキスト（見出しを解除） |
| `Ctrl+Enter` | タスクのチェックボックスを切り替え |

テキストを選択するとフローティングツールバー（太字、斜体、取り消し線、コード、ハイライト、リンク）が使え、レンダリングされた`[ ]`チェックボックスをクリックすると、どのモードでもタスクを切り替えられます。

## テキストの編集

| ショートカット | 動作 |
|---|---|
| `Ctrl+Z` / `Ctrl+Y`（または`Ctrl+Shift+Z`） | 元に戻す / やり直す |
| `Ctrl+A` | すべて選択 |
| `Tab` / `Shift+Tab` | インデント / インデント解除 |
| `Alt+↑` / `Alt+↓` | 行を上 / 下に移動 |
| `Shift+Alt+↓` | 行を複製 |
| `Ctrl+Shift+K` | 行を削除 |
| `Ctrl+D` | 次に一致する箇所を選択に追加 |
| `Ctrl+F` | ノート内で検索と置換 |
| `Ctrl+G` / `F3` | 次を検索（Shiftで前へ） |
| `Ctrl+Alt+G` | 行に移動 |
| `Ctrl+Shift+[` / `Ctrl+Shift+]` | 折りたたむ / 展開（macOS: `⌘⌥[` / `⌘⌥]`） |
| `Ctrl+Space` | オートコンプリートを表示 |

## 挿入

エディターに直接入力します:

| 入力 | 動作 |
|---|---|
| `/` | コマンドメニュー（ブロック、書式、テンプレート、絵文字） |
| `@` | メンション：日付、ノートへのリンク、またはデータベース |
| `[[` | Wikiリンク——ノートを選択 |
| `![[` | ノートまたは画像を埋め込む |
| `#` | タグを補完 |
| `:name` | 名前で絵文字を補完（文字を挿入） |
| `Ctrl+Alt+T` | テンプレートを挿入 |

エディターにファイルをドロップすると、画像は埋め込まれます（画像以外のファイルはコピーしてリンクされます）。

## グラフ

| ショートカット | 動作 |
|---|---|
| `Ctrl+Shift+G` | グラフ / 保管庫マップを開く |
| 矢印キー | ノード間でフォーカスを移動 |
| `Enter` | フォーカスしたノードを開く |

マウス：**スクロール**でズームします。**中クリックドラッグ**または`Ctrl`+ドラッグでマップを移動します（ノード上でも可）。**空いている場所をドラッグ**すると選択範囲の矩形が描かれます（保管庫マップ）。`Alt`+ドラッグで、ノードをリンクされた隣接ノードごと移動します（保管庫マップ）。**ノードを別のノードにドラッグ**すると、リンクまたはデータベースのリレーションを作成します（保管庫マップ）。シングルクリックは、コンテキストグラフ / データベースグラフでは開き、保管庫マップでは選択します。**フォルダーの丸をダブルクリック**すると展開します（枠をダブルクリックすると折りたたみます）。`Ctrl`+クリックで分割表示に開き、**中クリック**で新しいタブに開きます。**右クリック**でコンテキストメニューが開きます。**ピン留めアイコン**（右上）で、位置を記憶するかどうかを切り替えます。

## データベース（.base）

| 入力 | 動作 |
|---|---|
| セルをクリック | その場で編集、チェックボックスは直接切り替え |
| カード / 行をクリック | ノートをピークウィンドウで開く |
| カードを`Ctrl`+クリック | ノートを分割表示で開く |
| カードをドラッグ | ボード：列の値を設定、カレンダー/タイムライン：日付を設定 |
| 列ヘッダーをドラッグ | 列を並べ替え（プロパティの選択肢も含む） |
| テーブルヘッダーをクリック | 並べ替え順を変更 |
| ビューのピルをクリック / ドラッグ | ビューを切り替え、ドラッグで並べ替え、`▾`で名前変更/複製/削除 |
| **+ エントリー** | デフォルトテンプレートで新規エントリー（`▾` = テンプレート/フォルダー） |

## マウスとジェスチャー（どこでも）

- **ブロックハンドル（`⠿`）:** タップでブロックメニュー（変換、複製、移動、削除）、ドラッグで並べ替え。
- **タブ:** ドラッグで並べ替え、ペイン間へドラッグ、端へドラッグで分割。
- **リンク:** Wiki / Markdownリンクをクリックして開きます（`Ctrl`+クリックで新しいタブ）。
- **右クリック**で選択範囲やフィールドに対して、Plainva独自のコピー / 切り取り / 貼り付けメニューを開きます。`F5`と`Ctrl+R`は再読み込み**しません**（タブと未保存の内容が失われてしまうためです）。
- **ピークウィンドウ:** ヘッダーのドラッグで移動、角のグリップでサイズ変更。`◀ ▶`は専用の履歴です。
- **カレンダー:** 月名をクリックして月/年を選択、日をクリックしてそのデイリーノートを開きます。
- **画像エディター:** マウスで描画（ペン/矢印/長方形/テキスト/切り抜き）、ズームはボタンで行います。
