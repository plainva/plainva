# キーボードショートカット

最終更新: 2026-07-05

最も重要なキーの組み合わせを一覧にまとめました。アプリ内では、`F1`（または**設定 → キーボードショートカットを表示**）でいつでも同じ一覧を確認できます。macOSでは、`⌘`が`Ctrl`の代わりになります。

## グローバル

| ショートカット | 動作 |
|---|---|
| `Ctrl+O`または`Ctrl+K` | クイックスイッチャー——ファイルを検索して開く |
| `Ctrl+P` | **コマンド**——コマンドパレットを開く（入力してEnterで実行） |
| `Ctrl+Alt+B` | 左サイドバーを切り替え |
| `Ctrl+Alt+R` | 右サイドバーを切り替え |
| `Ctrl+,` | 設定を開く |
| `F1` | キーボードショートカットを表示 |
| `Esc` | 操作をキャンセル（ダイアログを閉じる、名前変更をキャンセル） |

## エディター

| ショートカット | 動作 |
|---|---|
| `/`（行頭で） | 書式設定メニュー（スラッシュメニュー）を開く |
| `@`（テキスト内で） | 日付、ノートへのリンク、またはデータベースを挿入 |
| `Ctrl+F` | 現在のノート内で検索と置換 |
| `Ctrl+Alt+T` | テンプレートを挿入 |
| `Ctrl+Alt+V` | エディターを分割——左右に並べる |
| `Ctrl+Alt+S` | エディターを分割——上下に並べる |

## ファイルツリー

| 入力 | 動作 |
|---|---|
| クリック | 選択 |
| `Ctrl`+クリック | 選択に追加/選択から削除 |
| `Shift`+クリック | 範囲を選択 |
| 中クリック | 新しいタブで開く |

## データベース (.base)

| 入力 | 動作 |
|---|---|
| セル/値をクリック | 直接編集（インライン編集） |
| エントリのタイトルをクリック | ピーク画面で開く |
| エントリを`Ctrl`+クリック | 分割表示で開く |
