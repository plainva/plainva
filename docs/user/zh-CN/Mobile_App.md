# 移动应用

更新日期：2026-07-12

Plainva也提供Android和iOS版应用。它使用与桌面版相同的Markdown文件、相同的**OKF**格式以及相同的同步引擎——你的仓库在两端保持完全一致。

## 布局

- **底部栏：** 最多可选择四个板块（笔记、今天、标签、书签、日历、数据库），围绕在固定的**＋**按钮周围。可在**设置** → **标签栏**中更改所选内容。
- **＋：** 轻点即可立即捕获一篇新笔记（保存到当前可见的文件夹，否则保存到收件箱文件夹）。长按可打开新建：笔记、每日笔记、文件夹、数据库、“从模板…”。
- **顶部栏：** 搜索和更多菜单；主页还会额外显示“最近”和你的书签。

## 阅读和编辑笔记

笔记以**渲染后的只读**状态打开；右上角的笔形图标可切换到编辑状态（键盘上方会出现工具栏：格式、列表、内部链接、斜杠命令、插入照片）。`![[笔记]]`嵌入会以可点击的预览卡片形式呈现。

**ⓘ** 符号会打开笔记的上下文面板：属性（可直接编辑）、反向链接、大纲、Markdown源码、笔记内搜索，以及**版本历史**——每次编辑都会自动生成快照，你可以查看、比较并恢复。

## 数据库（`.base`）

`.base`数据库的使用方式与桌面版相同：每一种视图（表格、列表、画廊、看板、日历、时间轴）、按类型编辑单元格、长按移动看板卡片。**配置**用于管理视图、列、筛选条件（包括分组）、排序和属性。关联模式（目标数据库、数量限制）仍需在桌面版中维护。

## 同步

在**更多** → **仓库**中连接云存储（WebDAV / Nextcloud、Google Drive、OneDrive、Dropbox、S3）。每个连接都会在设备上拥有各自独立的仓库。仓库页面会显示状态、进度、待处理的传输，并提供**导出仓库**（通过分享面板生成ZIP）。

## 安全保障

快照（版本历史）、草稿日志（发生崩溃后，笔记会提供你最后未保存的状态）以及带对比视图的冲突副本，共同保护你的数据。保留策略可在**设置**中配置。

## 分享与快捷方式（Android）

从其他应用分享的文本会作为一篇新笔记保存到收件箱文件夹。长按应用图标可使用**新笔记**和**今天**这两个快捷方式。
