# 移动应用

更新日期：2026-07-21

Plainva也提供Android和iOS版应用。它使用与桌面版相同的Markdown文件、相同的**OKF**格式以及相同的同步引擎——你的仓库在两端保持完全一致。

## 布局

- **底部栏：** 三个可自由排列的板块，加上固定的**更多**标签。**更多**中列出所有板块（笔记、今天、标签、书签、日历、数据库、关系图）——轻点即可打开，**手柄**用于调整列表顺序：排在最前的三项组成底部栏（以边框标出），将某项向上拖动即可将其纳入底部栏。
- **＋**以圆形按钮的形式悬浮在底部栏上方，点击即可打开快速新建：笔记、每日笔记、文件夹、数据库、“从模板…”。
- **顶部栏：** 搜索和**设置**（⋮）；主页还会额外显示“最近”和你的书签。
- **设置：** ⋮ 按钮会先打开区域列表（类似桌面版设置的左侧）——轻点即可打开相应页面。最上方的**当前仓库**会带你进入仓库管理：切换仓库（勾选标记 = 当前）、**创建 Vault**和**连接云端仓库**。

## 阅读和编辑笔记

笔记以**渲染后的只读**状态打开；右上角的笔形图标可切换到编辑状态（键盘上方会出现工具栏：格式、列表、内部链接、斜杠命令、插入照片）。`![[笔记]]`嵌入会以可点击的预览卡片形式呈现。

点击顶栏中的**笔记详情**按钮（位于书签和 ⋮ 菜单之间）会打开笔记的上下文面板：属性（可直接编辑）、反向链接、大纲、关系图，以及**版本历史**——每次编辑都会自动生成快照，你可以查看、比较并恢复。Markdown源码和笔记内搜索位于 ⋮ 菜单中。

## 数据库（`.base`）

`.base`数据库的使用方式与桌面版相同：每一种视图（表格、列表、画廊、看板、日历、时间轴）、按类型编辑单元格、长按移动看板卡片。**配置**用于管理视图、列、筛选条件（包括分组）、排序和属性。关联模式（目标数据库、数量限制）仍需在桌面版中维护。

**便签板**视图会以两列便签卡片的形式显示笔记：轻点可打开笔记，长按会显示操作菜单（固定、标签、颜色、删除），长按后拖动可重新排序，复选框可以直接在卡片上勾选。顶部的输入框用于记录一篇新笔记。提示：将数据库指向你的收件箱文件夹（**设置** → **内容与结构**），这样＋快捷新建的笔记以及从其他应用分享过来的文本就会直接出现在便签板上。

## 日历与活动

**日历**（底部标签或通过"更多"）以月视图网格显示你的每日笔记。右上角的时钟图标可打开**活动日历**，提供**日**、**3天**和**日程**视图——你连接的日历与桌面版使用相同的账户模型。点击某个活动会显示其详情；如果是邀请，你可以直接在此**接受**、标记为**待定**或**拒绝**。

在活动日历中点击齿轮图标即可管理账户：在设备上使用应用专用密码连接**CalDAV**（例如Fastmail、Nextcloud、iCloud）；Google和Microsoft则通过浏览器登录连接。你可以按账户显示或隐藏单个日历。

## 同步

在**设置**（⋮）中，**当前仓库**会带你进入仓库管理；在那里连接云存储（WebDAV / Nextcloud、Google Drive、OneDrive、Dropbox、S3）。**连接云端仓库**会把已有的云端仓库带到设备上；**创建 Vault**会先问**在此设备上**还是**在在线服务中**，然后选择起始结构（空白或PARA等模板）——在在线路径下，接下来会进行连接，云端的目标文件夹可以在选取面板中通过**新建文件夹**新建，结构会在首次同步时上传。首次启动（“连接云端仓库”）提供的是同样的选择——已有或新建云端仓库。每个连接都会在设备上拥有各自独立的仓库。仓库页面会显示状态、进度、待处理的传输，并提供**导出仓库**（通过分享面板生成ZIP）。

## 安全保障

快照（版本历史）、草稿日志（发生崩溃后，笔记会提供你最后未保存的状态）以及带对比视图的冲突副本，共同保护你的数据。保留策略可在**设置** → **备份与版本历史**中配置。

## 分享与快捷方式

在 Android 和 iOS 上，共享的文本和 URL 会成为收件箱文件夹中的新笔记；图片和文件会作为附件导入（每个文件最多 25 MB）。在 Android 上，长按应用图标还可使用**新笔记**和**今天**快捷方式。仓库页面可开启**同步设置**，并用密码短语安全地解锁或锁定加密仓库。
