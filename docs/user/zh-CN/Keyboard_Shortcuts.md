# 快捷键

更新日期：2026-07-15

按`F1`（或使用**命令面板 → 显示快捷键**）可随时打开快捷键窗口。它会把所有快捷键和鼠标手势归入不同区域，通过顶部的分类标签切换；配有一个覆盖所有区域的搜索框；并能自动识别你的平台：在Windows和Linux上显示`Ctrl`/`Alt`，在macOS上显示`⌘`/`⌥`。因此下面表格中的`Ctrl`在macOS上代表`⌘`，`Alt`代表`⌥`。

## 常规

| 快捷键 | 操作 |
|---|---|
| `Ctrl+P`（或`Ctrl+Shift+P`） | 打开命令面板 |
| `Ctrl+O` | 快速切换——搜索并打开文件 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+Plus` / `Ctrl+Minus` | 放大 / 缩小界面（80–150 %） |
| `Ctrl+0` | 将缩放重置为100 % |
| `Ctrl+S` | 确认保存（Plainva会自动保存） |
| `F1` | 显示此窗口 |
| `Esc` | 取消操作（关闭对话框、取消重命名） |

## 笔记与文件

| 快捷键 | 操作 |
|---|---|
| `Ctrl+N` | 在所选文件夹中新建笔记 |
| `Ctrl+Shift+D` | 打开今天的日记 |
| `F2` | 重命名当前笔记 |
| `Ctrl+Alt+Left` / `Ctrl+Alt+Right` | 后退 / 前进（导航历史） |

在文件树中：**点击**可选择并打开，`Ctrl`+点击可切换多选，`Shift`+点击可选择一个范围，**鼠标中键点击**可在新标签页中打开文件，**拖动**可移动文件，**右键点击**可打开右键菜单。

## 视图、窗口与标签页

| 快捷键 | 操作 |
|---|---|
| `Ctrl+E` | 切换阅读 ↔ 编辑 |
| `Ctrl+Shift+E` | 切换实时预览 ↔ Markdown源码 |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | 切换左侧 / 右侧边栏 |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | 分屏——左右并排 / 上下排列 |
| `Ctrl+Shift+G` | 打开关系图 / 仓库地图 |
| `Ctrl+Shift+F` | 在整个仓库中查找和替换 |
| `Ctrl+T` | 新建标签页（选择要打开的笔记） |
| `Ctrl+W` | 关闭标签页 |
| `Ctrl+Shift+T` | 重新打开最近关闭的标签页 |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | 下一个 / 上一个标签页（所有系统均为`Ctrl`） |
| `Ctrl+1` … `Ctrl+8` | 跳转到标签页1–8 |
| `Ctrl+9` | 跳转到最后一个标签页 |

专注模式（隐藏两侧边栏）可在命令面板中使用。

## 文本格式

适用于编辑器（实时预览或Markdown源码）：

| 快捷键 | 操作 |
|---|---|
| `Ctrl+B` / `Ctrl+I` | 粗体 / 斜体 |
| `Ctrl+Shift+S` | 删除线 |
| `Ctrl+Shift+H` | 高亮（`==标记==`） |
| `Ctrl+K` | 插入链接 / 将选中内容转为链接 |
| `Ctrl+Shift+1` / `2` / `3` | 一级 / 二级 / 三级标题 |
| `Ctrl+Shift+0` | 普通文本（取消标题） |
| `Ctrl+Enter` | 切换任务复选框 |

你也可以选中文本以调出悬浮工具栏（粗体、斜体、删除线、行内代码、高亮、链接），并在任意模式下点击渲染出的`[ ]`复选框来切换任务状态。

## 编辑文本

| 快捷键 | 操作 |
|---|---|
| `Ctrl+Z` / `Ctrl+Y`（或`Ctrl+Shift+Z`） | 撤销 / 重做 |
| `Ctrl+A` | 全选 |
| `Tab` / `Shift+Tab` | 缩进 / 取消缩进 |
| `Alt+Up` / `Alt+Down` | 将当前行上移 / 下移 |
| `Shift+Alt+Down` | 复制当前行 |
| `Ctrl+Shift+K` | 删除当前行 |
| `Ctrl+D` | 将下一个相同内容加入选区 |
| `Ctrl+F` | 在笔记中查找和替换 |
| `Ctrl+G` / `F3` | 查找下一个（Shift查找上一个） |
| `Ctrl+Alt+G` | 跳转到指定行 |
| `Ctrl+Shift+[` / `Ctrl+Shift+]` | 折叠 / 展开（macOS：`⌘⌥[` / `⌘⌥]`） |
| `Ctrl+Space` | 触发自动补全 |

## 插入

直接在编辑器中输入：

| 输入 | 操作 |
|---|---|
| `/` | 命令菜单（块、格式、模板、表情符号） |
| `@` | 提及：日期、笔记链接或数据库 |
| `[[` | 内部链接——选择一条笔记 |
| `![[` | 嵌入笔记或图片 |
| `#` | 补全标签 |
| `:name` | 按名称补全表情符号（插入实际字符） |
| `Ctrl+Alt+T` | 插入模板 |

把文件拖放到编辑器中会嵌入图片（其他文件则会被复制进来并添加链接）。

## 关系图

| 快捷键 | 操作 |
|---|---|
| `Ctrl+Shift+G` | 打开关系图 / 仓库地图 |
| 方向键 | 在节点之间移动焦点 |
| `Enter` | 打开聚焦的节点 |

鼠标操作：**滚轮**缩放；**按住鼠标中键拖动**或`Ctrl`+拖动可平移地图（即使在节点上方）；**在空白处拖动**会画出一个框选区域（仓库地图）；`Alt`+拖动可移动节点及其相连的邻居（仓库地图）；**将一个节点拖到另一个节点上**可创建链接或数据库关联（仓库地图）；单击可打开（上下文关系图/数据库关系图）或选择（仓库地图）；**双击文件夹气泡**将其展开（双击外环可折叠）；`Ctrl`+点击在拆分窗格中打开，**鼠标中键点击**在新标签页中打开；**右键点击**打开右键菜单；**图钉**（右上角）用于切换是否记住位置。

## 数据库（.base）

| 输入 | 操作 |
|---|---|
| 点击单元格 | 直接编辑；复选框可直接切换 |
| 点击卡片 / 行 | 在悬浮的速览窗口中打开笔记 |
| `Ctrl`+点击卡片 | 在拆分窗格中打开笔记 |
| 拖动卡片 | 看板：设置列值；日历/时间轴：设置日期 |
| 拖动列标题 | 调整列顺序（同时调整属性的选项顺序） |
| 点击列标题 | 更改排序方式 |
| 点击 / 拖动视图标签 | 切换视图；拖动可调整顺序；`▾`重命名/创建副本/删除 |
| **+ 条目** | 使用默认模板新建条目（`▾` = 模板/文件夹） |

## 鼠标与手势（随处可用）

- **块手柄（`⠿`）：**点击可打开块菜单（转换为、创建副本、移动、删除）；拖动可调整顺序。
- **标签页：**拖动可调整顺序、在窗格间移动，或拖到边缘进行拆分。
- **链接：**点击内部链接或Markdown链接即可打开（`Ctrl`+点击会在新标签页中打开）。
- **右键点击**选中内容或输入框，会打开Plainva自带的复制 / 剪切 / 粘贴菜单。`F5`和`Ctrl+R`**不会**重新加载（这样会关闭标签页并丢失未保存的内容）。
- **速览窗口：**拖动顶部可移动，拖动边角可调整大小；`◀ ▶`有独立的浏览历史。
- **日历：**点击月份名称可选择月/年；点击某一天可打开该日的日记。
- **图片编辑器：**用鼠标绘制（画笔/箭头/矩形/文字/裁剪）；通过按钮缩放。
