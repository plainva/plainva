# 快捷键

更新日期：2026-07-05

最重要的快捷键组合一览。在应用内，`F1`（或**设置 → 显示快捷键**）随时可以显示同样的列表。在macOS上，`⌘`代替`Ctrl`。

## 全局

| 快捷键 | 操作 |
|---|---|
| `Ctrl+O`或`Ctrl+K` | 快速切换——搜索并打开文件 |
| `Ctrl+P` | **命令**——打开命令面板（输入内容，回车执行） |
| `Ctrl+Alt+B` | 切换左侧边栏 |
| `Ctrl+Alt+R` | 切换右侧边栏 |
| `Ctrl+,` | 打开设置 |
| `F1` | 显示快捷键 |
| `Esc` | 取消操作（关闭对话框、取消重命名） |

## 编辑器

| 快捷键 | 操作 |
|---|---|
| `/`（在行首） | 打开格式菜单（斜杠菜单） |
| `@`（在正文中） | 插入日期、笔记链接或数据库 |
| `Ctrl+F` | 在当前笔记中查找和替换 |
| `Ctrl+Alt+T` | 插入模板 |
| `Ctrl+Alt+V` | 分屏——左右并排 |
| `Ctrl+Alt+S` | 分屏——上下排列 |

## 文件树

| 操作 | 效果 |
|---|---|
| 单击 | 选择 |
| `Ctrl`+单击 | 添加到选择范围/从中移除 |
| `Shift`+单击 | 选择一个区间 |
| 鼠标中键点击 | 在新标签页中打开 |

## 数据库（.base）

| 操作 | 效果 |
|---|---|
| 点击单元格/数值 | 直接编辑（内联编辑） |
| 点击条目标题 | 在预览窗口中打开 |
| `Ctrl`+点击条目 | 在分屏中打开 |
