# 搜索

更新日期：2026-07-15

Plainva提供三种搜索方式：跨整个仓库的全文搜索、用于打开文件的快速切换器，以及笔记内的查找和替换。

## 跨仓库全文搜索

侧边栏顶部的搜索框会搜索整个仓库——标题*和*内容。它由一个本地全文索引（SQLite FTS5）支撑，该索引会在仓库打开时构建，并在每次更改后保持最新；因此搜索可以离线进行，也不会有明显延迟。

搜索会随着你的输入即时响应：词语前缀已经可以匹配（输入"项目plan"中的"Projec"即可找到"Project plan"）——无需按Enter。字段右侧的**X**会清除当前搜索（也可以按`Esc`）；之后侧边栏会恢复显示正常的文件树。

结果列表顶部会显示命中次数，并对结果进行分组：**文件名**命中排在前面（搜索词出现在笔记的名称中），然后是**内容**命中。每一行都会显示文档图标、文件夹路径，以及——对于内容命中——一段带有高亮匹配内容的文本摘录。点击某个结果会打开该笔记，并直接跳转到第一处匹配位置；该处内容会被选中。如果没有任何匹配，列表会显示**无结果**。

搜索框同样适用于其他侧边栏视图：在**标签**视图中会筛选标签列表，在**书签**视图中会筛选书签。

### 搜索操作符

- `"精确短语"`——引号会精确匹配这个词语序列。对单个词来说，这也相当于全字匹配：`"plan"`能找到"plan"，但找不到"planning"。
- `-词语`——排除包含该词语的笔记（短语同样适用：`-"旧版本"`）。
- `path:文件夹`——只匹配路径中包含该文本的文件（例如`path:Projects`；包含空格时：`path:"My Folder"`）。
- `tag:名称`——只匹配带有该标签的笔记，包括嵌套标签：`tag:project`也能找到`#project/internal`。`tag:#project`同样有效。
- 操作符可以取反（`-path:Archive`、`-tag:done`），也可以与搜索词自由组合：`plan tag:project -draft`。
- 多个词语之间以AND方式组合。词语中的特殊字符（如`- ( ) : *`）不会造成问题——Plainva会按字面处理输入内容。

## 快速切换器

`Ctrl+O`或`Ctrl+K`会打开快速切换器：输入内容，用方向键导航，按`Enter`打开。不输入任何内容时，会显示**最近的文件**列表——这是在当前笔记之间跳转最快的方式。匹配项也可以直接在新标签页中打开（对话框底部会显示对应的按键）。

匹配采用模糊匹配：输入`prjplan`也能找到"Project Plan"——字母只需按顺序出现，且出现在词首的字母权重更高。当笔记尚不存在时，列表会显示**创建"…"**：按`Enter`会立即在仓库根目录创建并打开它——输入名称，按Enter，即可开始写作。

在名称命中结果下方，切换器还会显示一个**内容**分组：文本内容与输入匹配的笔记，并带有高亮的匹配摘录。打开这样的结果会直接跳转到笔记内的匹配位置——就像侧边栏搜索一样。

## 笔记内的查找和替换

`Ctrl+F`会打开编辑器的搜索栏（在实时预览和源码模式下均可用）：

- **查找**，按`Enter`/**下一个**和**上一个**在匹配项之间移动；**全部**会高亮所有匹配项。
- 选项：**区分大小写**、**全字匹配**、**正则**。
- **替换**：替换单个匹配项（**替换**）或**全部替换**。

### 跨整个仓库

`Ctrl/Cmd+Shift+F`（或在命令面板中使用**在仓库中查找并替换**）会一次性搜索所有笔记。输入搜索词，点击**查找**，匹配结果会按笔记分组显示，每条都附带一行上下文。输入替换内容，取消勾选你想跳过的笔记，然后**在N篇笔记中替换**会重写其余笔记——每篇笔记都会通过安全的方式写回（原子写入+版本快照），因此过时的预览内容永远不会覆盖更新的内容。区分大小写、全字匹配和正则表达式在这里同样适用；在正则模式下，替换内容中还可以使用`$1`/`$2`反向引用。

## 标签

侧边栏的**标签**视图列出仓库中所有的`#标签`及其命中次数；点击后会显示**带有#标签的文件**。标签既可以出现在正文中（`#project`），也可以出现在Frontmatter中（`tags: [project]`）。侧边栏的搜索框同样会筛选标签列表。

**重命名标签**可以在整个仓库范围内生效：在**标签**视图中右键点击某个标签，输入新名称。Plainva会在所有地方重写该标签——包括笔记正文中（`#tag`及其`#tag/child`子标签）以及Frontmatter中（`tags:`）——并通过同样安全的方式将每篇受影响的笔记写回。仅仅包含该名称的无关标签（例如`#area/tag`）不会受到影响。

## 在笔记内导航

右侧边栏的**大纲**列出当前笔记的所有标题——点击即可跳转到对应位置。在笔记之间跳转时，**反向链接**（谁链接到这里）以及编辑器的**后退**/**前进**按钮同样很有帮助。
