# Skróty klawiszowe

Stan na: 2026-07-10

Najważniejsze kombinacje klawiszy w jednym miejscu. W aplikacji `F1` (lub **Ustawienia → Pokaż skróty klawiszowe**) w każdej chwili pokazuje tę samą listę. Na macOS `⌘` zastępuje `Ctrl`.

## Globalne

| Skrót | Akcja |
|---|---|
| `Ctrl+O` lub `Ctrl+K` | Szybkie przełączanie – wyszukiwanie i otwieranie plików |
| `Ctrl+P` | **Polecenia** – otwórz paletę poleceń (wpisz, Enter uruchamia) |
| `Ctrl+Alt+B` | Przełącz lewy pasek boczny |
| `Ctrl+Alt+R` | Przełącz prawy pasek boczny |
| `Ctrl+,` | Otwórz ustawienia |
| `Ctrl+Plus` / `Ctrl+Minus` | Powiększenie interfejsu (80–150 %) |
| `Ctrl+0` | Resetuj powiększenie interfejsu |
| `F1` | Pokaż skróty klawiszowe |
| `Esc` | Anuluj akcję (zamknij okno dialogowe, przerwij zmianę nazwy) |

## Edytor

| Skrót | Akcja |
|---|---|
| `/` (na początku wiersza) | Otwórz menu formatowania (menu slash) |
| `@` (w tekście) | Wstaw datę, link do notatki lub bazę danych |
| `Ctrl+F` | Znajdź i zamień w bieżącej notatce |
| `Ctrl+Alt+T` | Wstaw szablon |
| `Ctrl+Alt+V` | Podziel edytor – obok siebie |
| `Ctrl+Alt+S` | Podziel edytor – jeden nad drugim |

## Drzewo plików

| Wejście | Akcja |
|---|---|
| Kliknięcie | Zaznacz |
| `Ctrl`+klik | Dodaj do zaznaczenia / usuń z zaznaczenia |
| `Shift`+klik | Zaznacz zakres |
| Kliknięcie środkowym przyciskiem | Otwórz w nowej karcie |

## Bazy danych (.base)

| Wejście | Akcja |
|---|---|
| Kliknięcie w komórkę/wartość | Edytuj bezpośrednio (edycja inline) |
| Kliknięcie tytułu wpisu | Otwórz w oknie podglądu |
| `Ctrl`+klik na wpis | Otwórz w podziale |
