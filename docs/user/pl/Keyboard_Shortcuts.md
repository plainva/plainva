# Skróty klawiszowe

Stan na: 2026-07-15

`F1` (lub **Paleta poleceń → Pokaż skróty klawiszowe**) w każdej chwili otwiera okno skrótów. Grupuje ono każdy skrót i gest myszy w obszary, między którymi przełącza się za pomocą chipów u góry, ma pole wyszukiwania obejmujące wszystkie obszary i automatycznie wykrywa platformę: pokazuje `Ctrl`/`Alt` w Windows i Linux oraz `⌘`/`⌥` w macOS. W tabelach poniżej `Ctrl` oznacza więc `⌘` w macOS, a `Alt` — `⌥`.

## Ogólne

| Skrót | Akcja |
|---|---|
| `Ctrl+P` (lub `Ctrl+Shift+P`) | Otwórz paletę poleceń |
| `Ctrl+O` | Szybkie przełączanie – wyszukiwanie i otwieranie plików |
| `Ctrl+,` | Otwórz ustawienia |
| `Ctrl+Plus` / `Ctrl+Minus` | Powiększ / pomniejsz interfejs (80–150 %) |
| `Ctrl+0` | Zresetuj powiększenie do 100 % |
| `Ctrl+S` | Potwierdź zapis (Plainva zapisuje automatycznie) |
| `F1` | Pokaż skróty klawiszowe |
| `Esc` | Anuluj akcję (zamknij okno dialogowe, przerwij zmianę nazwy) |

## Notatki i pliki

| Skrót | Akcja |
|---|---|
| `Ctrl+N` | Nowa notatka w wybranym folderze |
| `Ctrl+Shift+D` | Otwórz/utwórz dzisiejszą notatkę |
| `F2` | Zmień nazwę aktywnej notatki |
| `Ctrl+Alt+Lewo` / `Ctrl+Alt+Prawo` | Wstecz / Do przodu (historia nawigacji) |

W drzewie plików: **kliknięcie** wybiera i otwiera, `Ctrl`+klik przełącza zaznaczenie (wielokrotny wybór), `Shift`+klik zaznacza zakres, **kliknięcie środkowym przyciskiem** otwiera plik w nowej karcie, **przeciąganie** przenosi pliki, **kliknięcie prawym przyciskiem** otwiera menu kontekstowe, a `Delete` przenosi zaznaczenie do kosza.

## Widok, okna i karty

| Skrót | Akcja |
|---|---|
| `Ctrl+E` | Przełącz czytanie ↔ edycję |
| `Ctrl+Shift+E` | Przełącz podgląd na żywo ↔ źródło Markdown |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | Przełącz lewy / prawy panel boczny |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | Podziel edytor – obok siebie / jeden nad drugim |
| `Ctrl+Shift+G` | Otwórz graf / mapę sejfu |
| `Ctrl+Shift+F` | Znajdź i zamień w vaulcie |
| `Ctrl+T` | Nowa karta (wybierz notatkę do otwarcia) |
| `Ctrl+W` | Zamknij kartę |
| `Ctrl+Shift+T` | Otwórz ponownie ostatnio zamkniętą kartę |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Następna / poprzednia karta (`Ctrl` we wszystkich systemach) |
| `Ctrl+1` … `Ctrl+8` | Przejdź do karty 1–8 |
| `Ctrl+9` | Przejdź do ostatniej karty |

Tryb skupienia (oba panele boczne ukryte) jest dostępny w palecie poleceń.

## Formatowanie tekstu

Działa w edytorze (podgląd na żywo lub źródło Markdown):

| Skrót | Akcja |
|---|---|
| `Ctrl+B` / `Ctrl+I` | Pogrubienie / kursywa |
| `Ctrl+Shift+S` | Przekreślenie |
| `Ctrl+Shift+H` | Wyróżnienie (`==zaznaczenie==`) |
| `Ctrl+K` | Wstaw link / zamień zaznaczenie w link |
| `Ctrl+Shift+1` / `2` / `3` | Nagłówek 1 / 2 / 3 |
| `Ctrl+Shift+0` | Zwykły tekst (usuń nagłówek) |
| `Ctrl+Enter` | Przełącz pole wyboru zadania |

Zaznaczenie tekstu wywołuje też pływający pasek narzędzi (pogrubienie, kursywa, przekreślenie, kod, wyróżnienie, link), a kliknięcie wyrenderowanego pola wyboru `[ ]` przełącza zadanie w dowolnym trybie.

## Wstawianie

Wpisz bezpośrednio w edytorze:

| Wejście | Akcja |
|---|---|
| `/` | Menu formatowania (bloki, formatowanie, szablony, emoji) |
| `@` | Wzmianka: data, link do notatki lub baza danych |
| `[[` | Link wewnętrzny – wybierz notatkę |
| `![[` | Osadź notatkę lub obraz |
| `#` | Uzupełnij tag |
| `:name` | Emoji po nazwie (wstawia znak) |
| `Ctrl+Alt+T` | Wstaw szablon |

Przeciągnięcie pliku do edytora osadza obraz (inny plik zostaje skopiowany i połączony linkiem).

## Graf

| Skrót | Akcja |
|---|---|
| `Ctrl+Shift+G` | Otwórz graf / mapę sejfu |
| Strzałki | Przesuń fokus między węzłami |
| `Enter` | Otwórz węzeł w fokusie |

Mysz: **przewijanie kółkiem** zmienia powiększenie; **przeciąganie środkowym przyciskiem myszy** lub `Ctrl`+przeciąganie przesuwa widok (nawet nad węzłem); **przeciąganie po pustym miejscu** rysuje lasso zaznaczania (mapa sejfu); `Alt`+przeciąganie przenosi węzeł razem z połączonymi sąsiadami (mapa sejfu); **przeciągnięcie jednego węzła na drugi** tworzy link lub relację bazy danych (mapa sejfu); pojedyncze kliknięcie otwiera (graf kontekstowy / graf bazy danych) lub zaznacza (mapa sejfu); **podwójne kliknięcie bańki folderu** rozwija go (podwójne kliknięcie pierścienia zwija); `Ctrl`+klik otwiera notatkę w podziale, **kliknięcie środkowym przyciskiem** otwiera notatkę w nowej karcie; **kliknięcie prawym przyciskiem** otwiera menu kontekstowe; **igła przypięcia** (u góry po prawej) przełącza, czy pozycje są zapamiętywane.

## Bazy danych (.base)

| Wejście | Akcja |
|---|---|
| Kliknięcie komórki | Edytuj komórkę na miejscu; pole wyboru przełącza się od razu |
| Kliknięcie karty / wiersza | Otwórz notatkę w pływającym oknie podglądu |
| `Ctrl`+klik na kartę | Otwórz notatkę w podziale |
| Przeciąganie karty | Tablica: ustaw wartość kolumny; kalendarz/oś czasu: ustaw datę |
| Przeciąganie nagłówka kolumny | Zmień kolejność kolumn (i opcji właściwości) |
| Kliknięcie nagłówka tabeli | Zmień kolejność sortowania |
| Kliknięcie / przeciąganie przycisku widoku | Przełączaj widoki; przeciągnij, aby zmienić kolejność; `▾` = zmień nazwę/duplikuj/usuń |
| **+ Wpis** | Nowy wpis z domyślnym szablonem (`▾` = szablon/folder) |

## Mysz i gesty (wszędzie)

- **Uchwyt bloku (`⠿`):** dotknięcie otwiera menu bloku (przekształć, duplikuj, przenieś, usuń); przeciąganie zmienia kolejność.
- **Karty:** przeciąganie zmienia ich kolejność, przenosi je między panelami, a przeciągnięcie do krawędzi dzieli widok.
- **Linki:** kliknięcie linku wewnętrznego lub linku Markdown otwiera go (`Ctrl`+klik w nowej karcie).
- **Kliknięcie prawym przyciskiem** na zaznaczeniu lub polu otwiera własne menu kontekstowe Plainva: kopiuj / wytnij / wklej. `F5` i `Ctrl+R` **nie** odświeżają (utraciłyby karty i niezapisane bufory).
- **Okno podglądu:** przeciąganie nagłówka przesuwa je, uchwyt w rogu zmienia rozmiar; `◀ ▶` ma własną historię.
- **Kalendarz:** kliknięcie nazwy miesiąca wybiera miesiąc/rok; kliknięcie dnia otwiera jego notatkę dzienną.
- **Edytor obrazów:** rysowanie myszą (pisak/strzałka/prostokąt/tekst/kadrowanie); powiększenie za pomocą przycisków.
