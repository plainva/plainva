# Atalhos de Teclado

Última revisão: 2026-07-15

Pressione `F1` (ou **Paleta de comandos → Mostrar atalhos de teclado**) para abrir a janela de atalhos a qualquer momento. Ela agrupa todos os atalhos e gestos do mouse em áreas que você alterna pelos chips na parte superior, tem um campo de busca que abrange todas as áreas e detecta automaticamente a sua plataforma: mostra `Ctrl`/`Alt` no Windows e no Linux, e `⌘`/`⌥` no macOS. Nas tabelas abaixo, portanto, `Ctrl` significa `⌘` no macOS, e `Alt` significa `⌥`.

## Geral

| Atalho | Ação |
|---|---|
| `Ctrl+P` (ou `Ctrl+Shift+P`) | Abrir a paleta de comandos |
| `Ctrl+O` | Alternador rápido – buscar e abrir arquivos |
| `Ctrl+,` | Abrir configurações |
| `Ctrl+Plus` / `Ctrl+Minus` | Aumentar / diminuir o zoom da interface (80–150 %) |
| `Ctrl+0` | Redefinir o zoom para 100 % |
| `Ctrl+S` | Confirmar o salvamento (o Plainva salva automaticamente) |
| `F1` | Mostrar esta janela |
| `Esc` | Cancelar ação (fechar diálogo, cancelar renomeação) |

## Notas e arquivos

| Atalho | Ação |
|---|---|
| `Ctrl+N` | Nova nota na pasta selecionada |
| `Ctrl+Shift+D` | Abrir a nota diária de hoje |
| `F2` | Renomear a nota ativa |
| `Ctrl+Alt+Left` / `Ctrl+Alt+Right` | Voltar / avançar (histórico de navegação) |

Na árvore de arquivos: **clique** seleciona e abre, `Ctrl`+clique alterna uma seleção múltipla, `Shift`+clique seleciona um intervalo, **clique com o botão do meio** abre um arquivo em uma nova aba, **arrastar** move arquivos, **clique com o botão direito** abre o menu de contexto, e `Delete` move a seleção para a lixeira.

## Visualização, janelas e abas

| Atalho | Ação |
|---|---|
| `Ctrl+E` | Alternar leitura ↔ edição |
| `Ctrl+Shift+E` | Alternar visualização ao vivo ↔ código Markdown |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | Alternar barra lateral esquerda / direita |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | Dividir editor – lado a lado / empilhado |
| `Ctrl+Shift+G` | Abrir o grafo / mapa do vault |
| `Ctrl+Shift+F` | Localizar e substituir no vault |
| `Ctrl+T` | Nova aba (escolher uma nota para abrir) |
| `Ctrl+W` | Fechar aba |
| `Ctrl+Shift+T` | Reabrir a última aba fechada |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Próxima aba / aba anterior (`Ctrl` em qualquer sistema) |
| `Ctrl+1` … `Ctrl+8` | Ir para a aba 1–8 |
| `Ctrl+9` | Ir para a última aba |

O modo de foco (ambas as barras laterais ocultas) está disponível na paleta de comandos.

## Formatar texto

Funciona no editor (visualização ao vivo ou código Markdown):

| Atalho | Ação |
|---|---|
| `Ctrl+B` / `Ctrl+I` | Negrito / itálico |
| `Ctrl+Shift+S` | Tachado |
| `Ctrl+Shift+H` | Destacar (`==marcação==`) |
| `Ctrl+K` | Inserir link / transformar a seleção em um link |
| `Ctrl+Shift+1` / `2` / `3` | Título 1 / 2 / 3 |
| `Ctrl+Shift+0` | Texto normal (remove o título) |
| `Ctrl+Enter` | Alternar a caixa de seleção da tarefa |

Você também pode selecionar texto para abrir a barra flutuante (negrito, itálico, tachado, código, destaque, link), e clicar em uma caixa de seleção `[ ]` renderizada para alternar uma tarefa em qualquer modo.

## Editar texto

| Atalho | Ação |
|---|---|
| `Ctrl+Z` / `Ctrl+Y` (ou `Ctrl+Shift+Z`) | Desfazer / refazer |
| `Ctrl+A` | Selecionar tudo |
| `Tab` / `Shift+Tab` | Aumentar / diminuir o recuo |
| `Alt+Up` / `Alt+Down` | Mover a linha para cima / para baixo |
| `Shift+Alt+Down` | Duplicar linha |
| `Ctrl+Shift+K` | Excluir linha |
| `Ctrl+D` | Adicionar a próxima ocorrência à seleção |
| `Ctrl+F` | Localizar e substituir na nota |
| `Ctrl+G` / `F3` | Localizar o próximo (Shift para o anterior) |
| `Ctrl+Alt+G` | Ir para a linha |
| `Ctrl+Shift+[` / `Ctrl+Shift+]` | Recolher / expandir (macOS: `⌘⌥[` / `⌘⌥]`) |
| `Ctrl+Space` | Acionar o preenchimento automático |

## Inserir

Digite diretamente no editor:

| Entrada | Ação |
|---|---|
| `/` | Menu de comandos (blocos, formatação, modelos, emoji) |
| `@` | Menção: data, link de nota ou banco de dados |
| `[[` | Wiki link – escolher uma nota |
| `![[` | Incorporar uma nota ou imagem |
| `#` | Completar uma tag |
| `:name` | Emoji pelo nome (insere o caractere) |
| `Ctrl+Alt+T` | Inserir modelo |

Soltar um arquivo no editor incorpora uma imagem (ou copia e vincula outro arquivo).

## Grafo

| Atalho | Ação |
|---|---|
| `Ctrl+Shift+G` | Abrir o grafo / mapa do vault |
| Teclas de seta | Mover o foco entre os nós |
| `Enter` | Abrir o nó em foco |

Mouse: a **roda do mouse** dá zoom; **arrastar com o botão do meio** ou `Ctrl`+arrastar move o mapa (mesmo sobre um nó); **arrastar em área vazia** desenha um laço de seleção (mapa do vault); `Alt`+arrastar move um nó com seus vizinhos vinculados (mapa do vault); **arrastar um nó sobre outro** cria um link ou uma relação de banco de dados (mapa do vault); um clique simples abre (grafo de contexto/base) ou seleciona (mapa do vault); **clique duplo na bolha de uma pasta** a expande (clique duplo no anel para recolher); `Ctrl`+clique abre na divisão, **clique com o botão do meio** abre em uma nova aba; **clique com o botão direito** abre o menu de contexto; o **alfinete** (canto superior direito) alterna se as posições são lembradas.

## Bancos de dados (.base)

| Entrada | Ação |
|---|---|
| Clique em uma célula | Editar a célula diretamente; uma caixa de seleção alterna na hora |
| Clique em um cartão / linha | Abrir a nota em uma janela flutuante de espiada |
| `Ctrl`+clique em um cartão | Abrir a nota na divisão |
| Arrastar um cartão | Quadro: define o valor da coluna; calendário/linha do tempo: define a data |
| Arrastar o cabeçalho de uma coluna | Reordenar colunas (e as opções da propriedade) |
| Clique no cabeçalho da tabela | Alterar a ordenação |
| Clique / arraste uma pílula de visualização | Alternar entre visualizações; arraste para reordenar; `▾` renomeia/duplica/exclui |
| **+ Entrada** | Nova entrada com o modelo padrão (`▾` = modelo/pasta) |

## Mouse e gestos (em qualquer lugar)

- **Alça do bloco (`⠿`):** toque para abrir o menu do bloco (transformar em, duplicar, mover, excluir); arraste para reordenar.
- **Abas:** arraste para reordenar, entre painéis, ou até uma borda para dividir.
- **Links:** clique em um link wiki / Markdown para abri-lo (`Ctrl`+clique para uma nova aba).
- **Clique com o botão direito** em uma seleção ou campo para abrir o menu próprio do Plainva de copiar / recortar / colar. `F5` e `Ctrl+R` **não** recarregam (isso descartaria abas e buffers não salvos).
- **Janela de espiada:** arraste o cabeçalho para mover, a alça no canto para redimensionar; `◀ ▶` tem histórico próprio.
- **Calendário:** clique no nome do mês para escolher mês/ano; clique em um dia para sua nota diária.
- **Editor de imagem:** desenhe com o mouse (caneta/seta/retângulo/texto/cortar); zoom pelos botões.
