# Atalhos de Teclado

Stand: 2026-07-05

As combinações de teclas mais importantes em um só lugar. Dentro do app, `F1` (ou **Configurações → Mostrar atalhos de teclado**) mostra a mesma lista a qualquer momento. No macOS, `⌘` substitui `Ctrl`.

## Global

| Atalho | Ação |
|---|---|
| `Ctrl+O` ou `Ctrl+K` | Alternador rápido – buscar e abrir arquivos |
| `Ctrl+P` | **Comandos** – abrir a paleta de comandos (digite, Enter executa) |
| `Ctrl+Alt+B` | Alternar a barra lateral esquerda |
| `Ctrl+Alt+R` | Alternar a barra lateral direita |
| `Ctrl+,` | Abrir configurações |
| `F1` | Mostrar atalhos de teclado |
| `Esc` | Cancelar ação (fechar diálogo, cancelar renomeação) |

## Editor

| Atalho | Ação |
|---|---|
| `/` (no início de uma linha) | Abrir o menu de formatação (menu de barra) |
| `@` (dentro do texto) | Inserir uma data, link de nota ou banco de dados |
| `Ctrl+F` | Localizar e substituir na nota atual |
| `Ctrl+Alt+T` | Inserir modelo |
| `Ctrl+Alt+V` | Dividir editor – lado a lado |
| `Ctrl+Alt+S` | Dividir editor – empilhado |

## Árvore de arquivos

| Entrada | Ação |
|---|---|
| Clique | Selecionar |
| `Ctrl`+clique | Adicionar à / remover da seleção |
| `Shift`+clique | Selecionar um intervalo |
| Clique com o botão do meio | Abrir em uma nova aba |

## Bancos de Dados (.base)

| Entrada | Ação |
|---|---|
| Clique em uma célula/valor | Editar diretamente (edição inline) |
| Clique no título de um item | Abrir na janela de pré-visualização |
| `Ctrl`+clique em um item | Abrir na divisão |
