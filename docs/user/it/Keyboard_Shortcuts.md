# Scorciatoie da tastiera

Ultimo aggiornamento: 2026-07-15

Premi `F1` (oppure **Palette dei comandi → Mostra le scorciatoie da tastiera**) per aprire in qualsiasi momento la finestra delle scorciatoie. Raggruppa ogni scorciatoia e gesto del mouse in aree che puoi cambiare con i chip in alto, ha un campo di ricerca che copre tutte le aree e rileva automaticamente la tua piattaforma: mostra `Ctrl`/`Alt` su Windows e Linux e `⌘`/`⌥` su macOS. Nelle tabelle qui sotto, `Ctrl` indica quindi `⌘` su macOS e `Alt` indica `⌥`.

## Generale

| Scorciatoia | Azione |
|---|---|
| `Ctrl+P` (o `Ctrl+Shift+P`) | Apri la palette dei comandi |
| `Ctrl+O` | Selettore rapido – cerca e apri file |
| `Ctrl+,` | Apri le impostazioni |
| `Ctrl+Plus` / `Ctrl+Minus` | Aumenta / riduci lo zoom dell'interfaccia (80–150 %) |
| `Ctrl+0` | Ripristina lo zoom al 100 % |
| `Ctrl+S` | Conferma il salvataggio (Plainva salva automaticamente) |
| `F1` | Mostra questa finestra |
| `Esc` | Annulla l'azione (chiudi la finestra, annulla la rinomina) |

## Note e file

| Scorciatoia | Azione |
|---|---|
| `Ctrl+N` | Nuova nota nella cartella selezionata |
| `Ctrl+Shift+D` | Apri la nota giornaliera di oggi |
| `F2` | Rinomina la nota attiva |
| `Ctrl+Alt+Sinistra` / `Ctrl+Alt+Destra` | Indietro / Avanti (cronologia di navigazione) |

Nell'albero dei file: **clic** seleziona e apre, `Ctrl`+clic attiva/disattiva una selezione multipla, `Shift`+clic seleziona un intervallo, **clic centrale** apre un file in una nuova scheda, **trascinare** sposta i file, **clic destro** apre il menu contestuale e `Canc` sposta la selezione nel cestino.

## Vista, finestre e schede

| Scorciatoia | Azione |
|---|---|
| `Ctrl+E` | Alterna lettura ↔ modifica |
| `Ctrl+Shift+E` | Alterna anteprima dal vivo ↔ sorgente Markdown |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | Attiva/disattiva la barra laterale sinistra / destra |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | Dividi editor – affiancati / sovrapposti |
| `Ctrl+Shift+G` | Apri il grafo / la mappa del vault |
| `Ctrl+Shift+F` | Trova e sostituisci nel vault |
| `Ctrl+T` | Nuova scheda (scegli una nota da aprire) |
| `Ctrl+W` | Chiudi scheda |
| `Ctrl+Shift+T` | Riapri l'ultima scheda chiusa |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Scheda successiva / precedente (`Ctrl` su qualsiasi sistema operativo) |
| `Ctrl+1` … `Ctrl+8` | Vai alla scheda 1–8 |
| `Ctrl+9` | Vai all'ultima scheda |

La modalità focus (entrambe le barre laterali nascoste) si trova nella palette dei comandi.

## Formattazione testo

Funziona nell'editor (anteprima dal vivo o sorgente):

| Scorciatoia | Azione |
|---|---|
| `Ctrl+B` / `Ctrl+I` | Grassetto / Corsivo |
| `Ctrl+Shift+S` | Barrato |
| `Ctrl+Shift+H` | Evidenziazione (`==evidenziato==`) |
| `Ctrl+K` | Inserisci link / trasforma la selezione in un link |
| `Ctrl+Shift+1` / `2` / `3` | Titolo 1 / 2 / 3 |
| `Ctrl+Shift+0` | Testo normale (rimuovi il titolo) |
| `Ctrl+Enter` | Attiva/disattiva la casella dell'attività |

Puoi anche selezionare del testo per usare la barra fluttuante (grassetto, corsivo, barrato, codice, evidenziazione, link), oppure fare clic su una casella `[ ]` renderizzata per attivare/disattivare un'attività in qualsiasi modalità.

## Inserisci

Da digitare direttamente nell'editor:

| Input | Azione |
|---|---|
| `/` | Menu dei comandi (blocchi, formattazione, modelli, emoji) |
| `@` | Menzione: data, link a una nota o database |
| `[[` | Link interno – scegli una nota |
| `![[` | Incorpora una nota o un'immagine |
| `#` | Completa un tag |
| `:name` | Emoji per nome (inserisce il carattere) |
| `Ctrl+Alt+T` | Inserisci modello |

Rilasciare un file nell'editor incorpora un'immagine (oppure copia e collega un altro file).

## Grafo

| Scorciatoia | Azione |
|---|---|
| `Ctrl+Shift+G` | Apri il grafo / la mappa del vault |
| Tasti freccia | Sposta il focus tra i nodi |
| `Invio` | Apri il nodo con il focus |

Mouse: **scorri** per lo zoom; **trascinamento con il tasto centrale** o `Ctrl`+trascinamento per spostare la mappa (anche sopra un nodo); **trascinare su uno spazio vuoto** disegna un lazo di selezione (mappa del vault); `Alt`+trascinamento sposta un nodo insieme ai suoi vicini collegati (mappa del vault); **trascinare un nodo su un altro** crea un link o una relazione di database (mappa del vault); un clic singolo apre (grafo contestuale/database) oppure seleziona (mappa del vault); **doppio clic su una bolla di cartella** per espanderla (doppio clic sull'anello per comprimerla); `Ctrl`+clic apre nella vista divisa, **clic centrale** apre in una nuova scheda; **clic destro** per il menu contestuale; la **puntina** (in alto a destra) attiva/disattiva la memorizzazione delle posizioni.

## Database (.base)

| Input | Azione |
|---|---|
| Clic su una cella | Modifica la cella sul posto; una casella di controllo si attiva/disattiva direttamente |
| Clic su una voce / riga | Apri la nota in una finestra di anteprima |
| `Ctrl`+clic su una voce | Apri la nota nella vista divisa |
| Trascinamento di una voce | Bacheca: imposta il valore della colonna; calendario/cronologia: imposta la data |
| Trascinamento dell'intestazione di una colonna | Sposta le colonne (e le opzioni della proprietà) |
| Clic sull'intestazione di una tabella | Cambia l'ordinamento |
| Clic / trascinamento sulla pillola di una vista | Cambia vista; trascina per spostare; `▾` = rinomina/duplica/elimina |
| **+ Voce** | Nuova voce con il modello predefinito (`▾` = modello/cartella) |

## Mouse e gesti (ovunque)

- **Maniglia del blocco (`⠿`):** tocca per il menu del blocco (trasforma in, duplica, sposta, elimina); trascina per riordinare.
- **Schede:** trascina per riordinare, tra i riquadri, o su un bordo per dividere.
- **Link:** fai clic su un link interno / Markdown per aprirlo (`Ctrl`+clic per una nuova scheda).
- **Clic destro** su una selezione o un campo per il menu di copia / taglia / incolla proprio di Plainva. `F5` e `Ctrl+R` **non** ricaricano (perderesti le schede e i buffer non salvati).
- **Finestra di anteprima:** trascina l'intestazione per spostarla, l'angolo per ridimensionarla; `◀ ▶` hanno una cronologia propria.
- **Calendario:** fai clic sul nome del mese per scegliere mese/anno; fai clic su un giorno per la sua nota giornaliera.
- **Editor di immagini:** disegna con il mouse (penna/freccia/rettangolo/testo/ritaglio); zoom con i pulsanti.
