# Scorciatoie da tastiera

Stand: 2026-07-05

Le combinazioni di tasti più importanti in un colpo d'occhio. All'interno dell'app, `F1` (o **Impostazioni → Mostra le scorciatoie da tastiera**) mostra lo stesso elenco in qualsiasi momento. Su macOS, `⌘` sostituisce `Ctrl`.

## Globali

| Scorciatoia | Azione |
|---|---|
| `Ctrl+O` o `Ctrl+K` | Selettore rapido – cerca e apri file |
| `Ctrl+P` | **Comandi** – apri la palette dei comandi (digita, Invio esegue) |
| `Ctrl+Alt+B` | Attiva/disattiva la barra laterale sinistra |
| `Ctrl+Alt+R` | Attiva/disattiva la barra laterale destra |
| `Ctrl+,` | Apri le impostazioni |
| `F1` | Mostra le scorciatoie da tastiera |
| `Esc` | Annulla l'azione (chiudi la finestra, annulla la rinomina) |

## Editor

| Scorciatoia | Azione |
|---|---|
| `/` (a inizio riga) | Apri il menu di formattazione (menu slash) |
| `@` (nel testo) | Inserisci una data, un link a una nota o un database |
| `Ctrl+F` | Trova e sostituisci nella nota corrente |
| `Ctrl+Alt+T` | Inserisci modello |
| `Ctrl+Alt+V` | Dividi editor – affiancati |
| `Ctrl+Alt+S` | Dividi editor – sovrapposti |

## Albero dei file

| Input | Azione |
|---|---|
| Clic | Seleziona |
| `Ctrl`+clic | Aggiungi alla / rimuovi dalla selezione |
| `Shift`+clic | Seleziona un intervallo |
| Clic centrale | Apri in una nuova scheda |

## Database (.base)

| Input | Azione |
|---|---|
| Clic su una cella/valore | Modifica direttamente (modifica in linea) |
| Clic sul titolo di una voce | Apri nella finestra di anteprima |
| `Ctrl`+clic su una voce | Apri nella vista divisa |
