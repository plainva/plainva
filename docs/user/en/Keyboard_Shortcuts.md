# Keyboard Shortcuts

Last reviewed: 2026-07-05

The most important key combinations at a glance. Inside the app, `F1` (or **Settings → Show keyboard shortcuts**) shows the same list at any time. On macOS, `⌘` replaces `Ctrl`.

## Global

| Shortcut | Action |
|---|---|
| `Ctrl+O` or `Ctrl+K` | Quick switcher – search and open files |
| `Ctrl+P` | **Commands** – open the command palette (type, Enter runs) |
| `Ctrl+Alt+B` | Toggle the left sidebar |
| `Ctrl+Alt+R` | Toggle the right sidebar |
| `Ctrl+,` | Open settings |
| `F1` | Show keyboard shortcuts |
| `Esc` | Cancel action (close dialog, cancel renaming) |

## Editor

| Shortcut | Action |
|---|---|
| `/` (at the start of a line) | Open the formatting menu (slash menu) |
| `@` (within text) | Insert a date, note link or database |
| `Ctrl+F` | Find & replace in the current note |
| `Ctrl+Alt+T` | Insert template |
| `Ctrl+Alt+V` | Split editor – side by side |
| `Ctrl+Alt+S` | Split editor – stacked |

## File tree

| Input | Action |
|---|---|
| Click | Select |
| `Ctrl`+click | Add to / remove from the selection |
| `Shift`+click | Select a range |
| Middle-click | Open in a new tab |

## Databases (.base)

| Input | Action |
|---|---|
| Click a cell/value | Edit directly (inline editing) |
| Click an entry title | Open in the peek window |
| `Ctrl`+click an entry | Open in split |
