# Raccourcis clavier

Dernière mise à jour : 2026-07-05

Les combinaisons de touches les plus importantes en un coup d'œil. Dans l'application, `F1` (ou **Paramètres → Afficher les raccourcis clavier**) affiche la même liste à tout moment. Sur macOS, `⌘` remplace `Ctrl`.

## Global

| Raccourci | Action |
|---|---|
| `Ctrl+O` ou `Ctrl+K` | Sélecteur rapide – rechercher et ouvrir des fichiers |
| `Ctrl+P` | **Commandes** – ouvrir la palette de commandes (tapez, Entrée exécute) |
| `Ctrl+Alt+B` | Basculer la barre latérale gauche |
| `Ctrl+Alt+R` | Basculer la barre latérale droite |
| `Ctrl+,` | Ouvrir les paramètres |
| `F1` | Afficher les raccourcis clavier |
| `Esc` | Annuler l'action (fermer la boîte de dialogue, annuler le renommage) |

## Éditeur

| Raccourci | Action |
|---|---|
| `/` (en début de ligne) | Ouvrir le menu de mise en forme (menu slash) |
| `@` (dans le texte) | Insérer une date, un lien de note ou une base de données |
| `Ctrl+F` | Rechercher & remplacer dans la note actuelle |
| `Ctrl+Alt+T` | Insérer un modèle |
| `Ctrl+Alt+V` | Scinder l'éditeur – côte à côte |
| `Ctrl+Alt+S` | Scinder l'éditeur – superposé |

## Arborescence de fichiers

| Saisie | Action |
|---|---|
| Clic | Sélectionner |
| `Ctrl`+clic | Ajouter à / retirer de la sélection |
| `Shift`+clic | Sélectionner une plage |
| Clic central | Ouvrir dans un nouvel onglet |

## Bases de données (.base)

| Saisie | Action |
|---|---|
| Clic sur une cellule/valeur | Modifier directement (édition en ligne) |
| Clic sur le titre d'un élément | Ouvrir dans la fenêtre d'aperçu |
| `Ctrl`+clic sur un élément | Ouvrir dans la vue scindée |
