# Raccourcis clavier

Dernière mise à jour : 2026-07-15

Appuyez sur `F1` (ou **Palette de commandes → Afficher les raccourcis clavier**) pour ouvrir à tout moment la fenêtre des raccourcis. Elle regroupe tous les raccourcis et gestes de souris en zones que vous changez avec les pastilles en haut, propose un champ de recherche qui couvre toutes les zones, et détecte automatiquement votre plateforme : elle affiche `Ctrl`/`Alt` sous Windows et Linux, et `⌘`/`⌥` sous macOS. Dans les tableaux ci-dessous, `Ctrl` désigne donc `⌘` sous macOS, et `Alt` désigne `⌥`.

## Général

| Raccourci | Action |
|---|---|
| `Ctrl+P` (ou `Ctrl+Shift+P`) | Ouvrir la palette de commandes |
| `Ctrl+O` | Sélecteur rapide – rechercher et ouvrir des fichiers |
| `Ctrl+,` | Ouvrir les paramètres |
| `Ctrl+Plus` / `Ctrl+Minus` | Zoom avant / arrière sur l'interface (80–150 %) |
| `Ctrl+0` | Réinitialiser le zoom à 100 % |
| `Ctrl+S` | Confirmer l'enregistrement (Plainva enregistre automatiquement) |
| `F1` | Afficher cette fenêtre |
| `Esc` | Annuler l'action (fermer la boîte de dialogue, annuler le renommage) |

## Notes et fichiers

| Raccourci | Action |
|---|---|
| `Ctrl+N` | Nouvelle note dans le dossier sélectionné |
| `Ctrl+Shift+D` | Ouvrir la note quotidienne d'aujourd'hui |
| `F2` | Renommer la note active |
| `Ctrl+Alt+Gauche` / `Ctrl+Alt+Droite` | Retour / Avancer (historique de navigation) |

Dans l'arborescence de fichiers : le **clic** sélectionne et ouvre, `Ctrl`+clic bascule la sélection (sélection multiple), `Shift`+clic sélectionne une plage, le **clic central** ouvre un fichier dans un nouvel onglet, le **glisser** déplace des fichiers, le **clic droit** ouvre le menu contextuel, et `Suppr` déplace la sélection vers la corbeille.

## Affichage, fenêtres et onglets

| Raccourci | Action |
|---|---|
| `Ctrl+E` | Basculer lecture ↔ édition |
| `Ctrl+Shift+E` | Basculer aperçu en direct ↔ source Markdown |
| `Ctrl+Alt+B` / `Ctrl+Alt+R` | Basculer la barre latérale gauche / droite |
| `Ctrl+Alt+V` / `Ctrl+Alt+S` | Scinder l'éditeur – côte à côte / superposé |
| `Ctrl+Shift+G` | Ouvrir le graphe / la carte du coffre |
| `Ctrl+Shift+F` | Rechercher et remplacer dans le vault |
| `Ctrl+T` | Nouvel onglet (choisir une note à ouvrir) |
| `Ctrl+W` | Fermer l'onglet |
| `Ctrl+Shift+T` | Rouvrir le dernier onglet fermé |
| `Ctrl+Tab` / `Ctrl+Shift+Tab` | Onglet suivant / précédent (`Ctrl` sur tous les systèmes, y compris macOS) |
| `Ctrl+1` … `Ctrl+8` | Aller à l'onglet 1–8 |
| `Ctrl+9` | Aller au dernier onglet |

Le mode focus (les deux barres latérales masquées) se trouve dans la palette de commandes.

## Mise en forme du texte

Fonctionne dans l'éditeur (aperçu en direct ou source) :

| Raccourci | Action |
|---|---|
| `Ctrl+B` / `Ctrl+I` | Gras / italique |
| `Ctrl+Shift+S` | Barré |
| `Ctrl+Shift+H` | Surlignage (`==marque==`) |
| `Ctrl+K` | Insérer un lien / transformer la sélection en lien |
| `Ctrl+Shift+1` / `2` / `3` | Titre 1 / 2 / 3 |
| `Ctrl+Shift+0` | Texte normal (supprimer le titre) |
| `Ctrl+Entrée` | Basculer la case à cocher de la tâche |

Vous pouvez aussi sélectionner du texte pour afficher la barre d'outils flottante (gras, italique, barré, code, surlignage, lien), et cliquer sur une case à cocher `[ ]` affichée pour basculer une tâche dans n'importe quel mode.

## Insérer

À taper directement dans l'éditeur :

| Saisie | Action |
|---|---|
| `/` | Menu de commandes (blocs, mise en forme, modèles, emoji) |
| `@` | Mention : date, lien de note ou base de données |
| `[[` | Lien interne – choisir une note |
| `![[` | Intégrer une note ou une image |
| `#` | Compléter un tag |
| `:name` | Emoji par nom (insère le caractère) |
| `Ctrl+Alt+T` | Insérer un modèle |

Déposer un fichier dans l'éditeur intègre une image (ou copie et lie un autre fichier).

## Graphe

| Raccourci | Action |
|---|---|
| `Ctrl+Shift+G` | Ouvrir le graphe / la carte du coffre |
| Touches fléchées | Déplacer le focus entre les nœuds |
| `Entrée` | Ouvrir le nœud focalisé |

Souris : la **molette** zoome ; le **clic molette + glisser** ou `Ctrl`+glisser déplace la carte (même au-dessus d'un nœud) ; **glisser sur une zone vide** dessine un rectangle de sélection (carte du coffre) ; `Alt`+glisser déplace un nœud avec ses voisins liés (carte du coffre) ; **glisser un nœud sur un autre** crée un lien ou une relation de base de données (carte du coffre) ; un **simple clic** ouvre (graphe contextuel / graphe de base) ou sélectionne (carte du coffre) ; **double-cliquer sur une bulle de dossier** le déplie (double-clic sur l'anneau pour le replier) ; `Ctrl`+clic ouvre dans la vue scindée, le **clic central** ouvre dans un nouvel onglet ; le **clic droit** ouvre le menu contextuel ; l'**épingle** (en haut à droite) active ou désactive la mémorisation des positions.

## Bases de données (.base)

| Saisie | Action |
|---|---|
| Cliquer sur une cellule | Modifier la cellule sur place ; une case à cocher bascule directement |
| Cliquer sur une carte / ligne | Ouvrir la note dans une fenêtre d'aperçu flottante |
| `Ctrl`+clic sur une carte | Ouvrir la note dans la vue scindée |
| Glisser une carte | Kanban : définir la valeur de la colonne ; calendrier/chronologie : définir la date |
| Glisser un en-tête de colonne | Réorganiser les colonnes (et les options de la propriété) |
| Cliquer sur un en-tête de tableau | Changer l'ordre de tri |
| Cliquer / glisser une pastille de vue | Changer de vue ; glisser pour réorganiser ; `▾` = renommer/dupliquer/supprimer |
| **+ Entrée** | Nouvelle entrée avec le modèle par défaut (`▾` = modèle/dossier) |

## Souris et gestes (partout)

- **Poignée de bloc (`⠿`) :** toucher pour le menu de bloc (transformer en, dupliquer, déplacer, supprimer) ; glisser pour réorganiser.
- **Onglets :** glisser pour réorganiser, entre les volets, ou vers un bord pour scinder.
- **Liens :** cliquer sur un lien interne / Markdown pour l'ouvrir (`Ctrl`+clic pour un nouvel onglet).
- **Clic droit** sur une sélection ou un champ pour le menu copier / couper / coller propre à Plainva. `F5` et `Ctrl+R` ne rechargent **pas** l'application (cela ferait perdre les onglets et les brouillons non enregistrés).
- **Fenêtre d'aperçu :** en-tête pour déplacer, angle pour redimensionner ; `◀ ▶` est son propre historique.
- **Calendrier :** cliquer sur le nom du mois pour choisir le mois ou l'année ; cliquer sur un jour pour sa note quotidienne.
- **Éditeur d'image :** dessiner avec la souris (crayon/flèche/rectangle/texte/rognage) ; zoomer avec les boutons.
