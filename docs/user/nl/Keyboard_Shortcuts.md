# Sneltoetsen

Laatst bijgewerkt: 2026-07-10

De belangrijkste toetscombinaties in één oogopslag. Binnen de app toont `F1` (of **Instellingen → Sneltoetsen tonen**) altijd dezelfde lijst. Op macOS vervangt `⌘` `Ctrl`.

## Algemeen

| Sneltoets | Actie |
|---|---|
| `Ctrl+O` of `Ctrl+K` | Snelkiezer – bestanden zoeken en openen |
| `Ctrl+P` | **Opdrachten** – de opdrachtenpalet openen (typen, Enter voert uit) |
| `Ctrl+Alt+B` | Linkerzijbalk tonen/verbergen |
| `Ctrl+Alt+R` | Rechterzijbalk tonen/verbergen |
| `Ctrl+,` | Instellingen openen |
| `Ctrl+Plus` / `Ctrl+Minus` | Interfacezoom vergroten/verkleinen (80–150 %) |
| `Ctrl+0` | Interfacezoom herstellen |
| `F1` | Sneltoetsen tonen |
| `Esc` | Actie annuleren (dialoogvenster sluiten, hernoemen annuleren) |

## Editor

| Sneltoets | Actie |
|---|---|
| `/` (aan het begin van een regel) | Het opmaakmenu openen (slash-menu) |
| `@` (in de tekst) | Een datum, notitielink of database invoegen |
| `Ctrl+F` | Zoeken & vervangen in de huidige notitie |
| `Ctrl+Alt+T` | Sjabloon invoegen |
| `Ctrl+Alt+V` | Editor splitsen – naast elkaar |
| `Ctrl+Alt+S` | Editor splitsen – boven elkaar |

## Bestandsboom

| Invoer | Actie |
|---|---|
| Klik | Selecteren |
| `Ctrl`+klik | Toevoegen aan / verwijderen uit de selectie |
| `Shift`+klik | Een bereik selecteren |
| Middelklik | Openen in een nieuw tabblad |

## Databases (.base)

| Invoer | Actie |
|---|---|
| Klik op cel/waarde | Direct bewerken (inline-editing) |
| Klik op de titel van een item | Openen in het peek-venster |
| `Ctrl`+klik op een item | Openen in split |
